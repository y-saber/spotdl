Нагло воруем музыку из Spotify

а то хули они?
  1. Ставим Python. Скачать можно тут
    Важно!
    Сразу в начале установки нужно поставить чекбокс Add Python 3.10 to PATH
  2. На всякий случай ставим свежий VC Redist. Взять можно здесь
  3. Далее создаем какую-нибудь папку, в которую мы будем качать музыку
    Например: C:\spotdl\
  4. Последнее что нужно скачать - ffmpeg. Взять можно здесь
    Сам ffmpeg нужно распаковать в какую-нибудь папку, для простоты туда же куда будем качать музыку. Например: C:\spotdl\ffmpeg\
    Отсюда нам нужен будет полный путь до exe файла, то есть: C:\spotdl\ffmpeg\bin\ffmpeg.exe
  5. Далее открываем cmd от имени админа
    Жмем Win -> пишем cmd -> ПКМ -> Запуск от админа
  6. В cmd нужно перейти в папку в которой мы будем работать и в которую будем качать музыку, то есть: C:\spotdl\
    В cmd пишем: cd C:\spotdl\
  7. Далее пишем: pip install spotdl и ждем
  8. Когда будет готово можем качать музыку
    Главное не забыть про путь до ffmpeg
    Общий формат команды такой:
    spotdl --ffmpeg <путь до ffmpeg> <ссылка на Spotify>
    Пример полной команды:
    spotdl --ffmpeg C:\spotdl\ffmpeg\bin\ffmpeg.exe https://open.spotify.com/playlist/0otUzePuYwSa4LXVYQ1bOF
  9. Музыка появится в этой же папке, то есть в C:\spotdl\
    Общий алгоритм работы spotdl примерно такой:
    • Берет список треков из ссылки на Spotify. Ссылка может быть на трек, на альбом, на артиста, на плейлист(не приватный)
    • По этому списку трек ищется на Youtube.Music
      Дополнительно к артисту-названию проверяется и длительность, для того чтобы Youtube.Music не подсунул чего-то не того в своем поиске
    • Трек скачивается с Youtube и конвертируется в аудио через ffmpeg
